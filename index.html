Inloggningssida

Utveckling // Design

En simpel inloggningssida med rörlig bakgrund, sidan är skriven i endast html och css. Videobakgrunden är implementerad med hjälp av HTML5 taggen 'video' och sedan justerad i css. Observera att ingen backend finns tillgänglig, utan det är endast frontend. Sidan var mest ett test av videobakgrunden.

Att-göra-lista

Utveckling // Design // funktionalitet

En att-göra-lista som hjälper användaren att hålla reda på saker. Backenden är skriven i vanilla javascript och objekten sparas lokalt på enheten så att allting finns kvar när man öppnar sidan/"appen" igen.
